> Hexo桌面客户端

联系我：[http://www.mspring.org](http://www.mspring.org)

### 功能简介
- 文章添加
- 文章修改
- 文章删除
- 文章支持七牛图片上传

### Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:9080
npm run dev

# build electron application for production
npm run build


# lint all JS/Vue component files in `src/`
npm run lint

```
